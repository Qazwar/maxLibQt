Detailed Description

The ActionPushButton class is a QPushButton which takes a default QAction, just like a QToolButton can.

Like QToolButton, it will inherit all properties from the default QAction, such as text, icon, checkable status & state, tool tip, and so on. The default action can be set with setDefaultAction() and retrieved with defaultAction(). The default action can also be set using the dedicated ActionPushButton(QAction *, QWidget *) constructor.

It also adds a triggered(QAction *) signal for all QActions added to the button (not just the default one).

void ActionPushButton::setDefaultAction ( QAction *  action)
slot

Sets the default action to action.

The action is added to the widget if it hasn't been already. To clear the default action, pass a nullptr as the action. Clearing the default action in this way does not remove it from this widget itself. Use QWidget::removeAction() for that instead, which will also clear the default action on this button (if the action being removed is the current default, of course).
