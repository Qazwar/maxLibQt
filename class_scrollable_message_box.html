Detailed Description

A simple message box with a large scrollable area (a QTextEdit ) for detailed text (including HTML formatting).

Version
1.0.0

This is a basic message box dialog for presenting an amount of text which would not comfortably fit in a regular popup message dialog. It addresses the deficiencies found in QMessageBox with "informative text" option, namely its small and fixed size and lack of text formatting options (no rich text or even font control).

Basic usage is like QMessageBox. You can set the dialog title, an optional short text message to show at the top, and the larger text block to show as the "details." There is only one "OK" button to dismiss the dialog.

When first shown, the dialog will try to esablish a reasonable size based on the contents of the QTextEdit. The dialog is also fully resizable.

Developers have direct access to the QTextEdit used for displaying the text, the QLineEdit used for the short text message, and the QDialogButtonBox with the default "Ok" button. There are a couple convenience functions for setting the font style and word wrap on the QTextEdit. By default a variable-width font is used with word-wrap on.
