Detailed Description

The BuddyLabel class is a QLabel with enhanced "buddy" capabilities.

It overrides the QLabel::setBuddy() method and, besides the usual shortcut handling provided by QLabel, it adds mouse click handling and mirroring of the buddy's tool tip text.

Mouse clicks are connected to the QWidget::setFocus slot. For QCheckBox it also connects to the click() slot so the box can be (un)checked by clicking on the label. Mouse double-clicks are connected to QLineEdit::selectAll() on widgets which either are or have a QLineEdit (like QAbstractSpinBox and editable QComboBox). Custom connections could be added by connecting to the clicked() and/or doubleClicked() signals, or inheriting and overriding the connectBuddy() virtual method.
